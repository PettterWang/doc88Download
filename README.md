说明

道客巴巴的文档在手机端有三种 

1 没有手机端（那用我的代码肯定爬不了） 

2 一页是一张gif     

3 一页由四张gif组合而成



第2种方式 可以尝试一下使用doc88.py



第3种的先用downloadsmallgif.py 下载小图片然后用imgswitch.py对图片进行拼接。

注意：downloadsmallgif.py和imgswitch.py仅为简单演示 并没有创建文件夹等操作并且imgswitch.py也仅仅合并了第一页的gif，后续合并其他页的gif和合并pdf请自便。
